## 📌 Overview

The **AI Visa Assistant** is a Streamlit-based web application designed to help users check their visa eligibility for multiple countries, including:

- 🇺🇸 USA  
- 🇬🇧 UK  
- 🇨🇦 Canada  
- 🇦🇺 Australia  
- 🇩🇪 Germany  

The system evaluates user inputs such as academic background, financial status, and supporting documents to generate:

- ✅ Eligibility Decision  
- 📊 Approval Probability  
- 📄 Professional Explanation  
- ⚠️ Risk Factors  
- ✔️ Positive Factors  

---

## ✨ Features

- 👤 Multi-step workflow (Personal → Academic → Application → Result)
- 🎯 Smart eligibility prediction logic
- 📊 Approval probability calculation (0–100%)
- 🧠 AI-style professional explanation
- 💬 Built-in assistant for:
  - Required documents  
  - Visa process  
  - Eligibility explanation  
- 🎨 Modern UI using Streamlit
- 🔁 Dynamic navigation using session state

---

## 🧠 How It Works

1. User enters **personal information**
2. Provides **academic & professional details**
3. Fills **visa-specific application inputs**
4. System applies **rule-based scoring**
5. Generates:
   - Eligibility status  
   - Approval probability  
   - Detailed explanation  
   - Strengths & risk factors
