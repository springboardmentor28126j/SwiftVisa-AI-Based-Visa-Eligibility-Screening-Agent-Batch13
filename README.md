# SwiftVisa AI - Visa Eligibility Checker

## Overview

This project is an AI-based visa eligibility screening system that evaluates user eligibility based on visa policies.

## Features

- User-friendly Streamlit interface
- Policy-based eligibility evaluation
- Supports multiple countries and visa types

## Tech Stack

- Python
- Streamlit
- JSON Dataset

## How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run:
   streamlit run streamlit_app.py

## Note

The system is designed to support LLM-based reasoning and can be extended using Gemini/OpenAI APIs.
